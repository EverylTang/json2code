# JSON to Code Converter

> Convert JSON to TypeScript, Go, Python, Java, Rust, C#, Swift, and Kotlin type definitions instantly.

![Next.js](https://img.shields.io/badge/Next.js-16-black)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🚀 Features

- **11 Languages Supported**: TypeScript, Go, Python, Java, Rust, C#, Swift, Kotlin, Ruby, PHP, Dart
- **Real-time Conversion**: See results as you type
- **Customizable Options**:
  - Root type name
  - Optional/nullable types
  - Serialization annotations
  - Type aliases vs interfaces
  - Lombok support for Java
  - Custom tag styles for Go
- **Import/Export**:
  - File upload/download
  - URL import
  - Share via link
- **Dark Mode**: Full dark/light theme support
- **i18n**: Chinese and English support
- **Code Editor**: Monaco Editor with syntax highlighting
- **Error Detection**: Line and column numbers for JSON errors

## 🎯 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:3000`

## 📦 Deployment

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

### Deploy to Netlify

```bash
# Install Netlify CLI
npm i -g netlify-cli

# Deploy
netlify deploy --prod
```

## 🛠️ Technology Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Editor**: Monaco Editor
- **Theme**: next-themes
- **Icons**: Lucide React

## 📖 Usage

1. Paste your JSON in the left editor
2. Select target language
3. View generated code in the right editor
4. Click "Copy" to copy to clipboard
5. Click "Download" to save as file
